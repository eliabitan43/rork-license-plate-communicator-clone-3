# Redesign the home page with a friendlier hero and top plate entry

## Features
- [x] Put the plate number entry at the top of the home screen inside the main welcome section.
- [x] Make the plate entry the clearest first action, with a simple path into the full message screen.
- [x] Keep quick actions close to the top so common reasons for messaging feel fast and familiar.
- [x] Balance the screen with recent activity, helpful shortcuts, and trust-building community details.
- [x] Show the driver’s own car details in a lighter supporting spot so the target plate input stays primary.

## Design
- [x] Use a friendlier, community-led visual style with softer blues, warm neutrals, and subtle teal accents.
- [x] Redesign the top area into a more welcoming hero with stronger hierarchy, cleaner spacing, and a more mobile-native feel.
- [x] Turn the plate number entry into a standout card that feels obvious, safe, and easy to use.
- [x] Use softer surfaces, better depth, rounded shapes, and gentle motion instead of heavy blocks.
- [x] Improve section rhythm so the screen feels calmer, more premium, and less crowded.
- [x] Incorporate the generated driver background image into the home hero with a readable cinematic overlay.

## Pages / Screens
- [x] **Home screen**: Welcome hero, top plate number field, quick send entry, community highlights, and recent activity.
- [x] **Message flow**: After entering a plate number, move directly into the full compose screen.
- [x] **Profile / vehicle area**: Surface the user’s own plate as a secondary reference, not the main home action.

## Plate Number Placement
- [x] Add the target plate number field directly inside the hero near the top of the home page.
- [x] Keep the user’s own plate visible as a smaller badge or card below the hero, so people do not confuse the two.
- [x] Use clear labels so one field is for messaging another driver and the other is for the user’s registered vehicle.
